[RISC-V]GD32VF103CBT6 开发环境搭建

1. 芯片参数

主频： 108MHz

SRAM: 32KB

ROM : 128KB

相比于 stm32f103cbt6(72MHz, 20KB SRAM, 128KB ROM), riscv 的硬件更胜一筹

2. 基于 Arduino 开发模式的环境搭建步骤

  1. 安装 VS Code

  2. 在 VS Code 中安装 platform IO 插件

  3. 在 platform IO 界面坐下间点击"New Terminal", 输入 github 网址同步代码到本地，以下地址二选一(第一个连接是稳定 release 版本，第二个链接是开发版本，实时更新的)

  4. 进入 platform IO 主页，点击 project examples，导入 arduino 项目即可

  1. 项目例子导入后，打开 platformio.ini 文件，可自行设置串口通信波特率，固件上传方式及调试工具(调试工具支持的比较多，可以支持 Jlink, FTDI 的 USB 转串口， RV-debugger 等等)

  1. 对项目进行编译，编译成功之后如下图

  1. 固件下载

    固件下载无需其它工具，芯片内部集成了 DFU 模式，可以通过配置 platformio.ini 中下载方式为 DFU, 另外需要注意，DFU 模式与电脑端通信是只支持 libusb 驱动，可以使用 zadig 软件进行 USB 驱动转换
