[PCB Layout]DRC及规则

1. DRC检查项设置

2. Design Rule设置

嘉立创PCB工艺参数

2.1 过孔

  1. 双面板：内径 >= 0.3mm, 外径 >= 0.5mm
  2. 多层板：内径 >= 0.2mm, 外径 >= 0.4mm

2.2 线宽

  1. 双面板：d >= 5mil
  2. 多层板：d >= 3.5mil

2.3 导线之间的间距(4mil <= d <= 10mil)

  1. 双面板：d >= 5mil (建议设置为最小 6mil)
  2. 多层板：d >= 3.5mil

2.4 过孔与走线之间的间距

d >= 6mil

2.5 焊盘边缘与走线之间的间距

d >= 6mil

2.6 字符大小

  1. 最小字符宽度：w >= 6mil
  2. 最小字符高度：h >= 40mil

2.7 最小工艺边

  1. 最小工艺边 >= 3mm

2.8 半孔工艺

  1. 最小孔径 >= 0.6mm

2.9 走线和焊盘与板边距离

  1. 单片出货：>= 0.3mm
  2. 拼版出货：>= 0.4mm (考虑拼版 v-cut)

2.10 阻焊层开窗

  1. 绿油桥 >= 0.05mm

2.11 铜厚

  1. 成品外层铜厚：1oz = 0.035mm
  2. 成品内层铜厚：0.5oz = 0.017mm
