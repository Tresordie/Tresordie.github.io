FreeRTOS 定时器组

1. FreeRTOS 的定时器组含义及特点

  1. 实质是软件定时器，不需要使用硬件定时器
  2. 实现方式是基于 FreeRTOS 的系统时钟节拍
  3. 可以创建很多个，且实现的功能与硬件定时器类似
  4. 通过定时器定时到达后的回调函数实现功能

2. FreeRTOS 中定时器组模式

  1. 单次模式
  2. 周期模式

3. 定时器任务实现机理

  1. 用户应用代码中调用定时器组 API 函数
  2. API 函数通过消息队列(Timer command queue)给定时器任务发送消息
  3. 在定时器任务里面执行实际操作
