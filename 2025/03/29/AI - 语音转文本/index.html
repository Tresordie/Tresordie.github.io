语音转文本

发表于2025-03-29|更新于2025-03-29|tool
|阅读量:

为什么要写这篇文章呢？由于最近工作中琐事太多，通过使用网页版ChatGPT 来提高效率，只使用了两次语音转文本之后就提示我要升级会员，否则只能等到第二天才能再次使用，所以我考虑是否可以部署一个本地且免费的语音转文本的软件，这样子不会存在次数受限的问题

🔴 whisper本地部署(免费)

  1. Python安装，3.7以上版本

  2. PyTorch安装Pytorch，Windows Stable version

  3. Chocolatey安装 chocolatey

  4. 使用Chocolatey安装ffmpeg,

  5. whisper安装

  6. 语音转文本

  7. 如何卸载

🔴 Google colaboratory(免费)

  1. 在Google drive中安装Google colaboratory

  2. 启动Google colaboratory，重新命名且设置运行时为GPU

  3. 安装whisper, ffmpeg
    !pip install git+https://github.com/openai/whisper.git

    !sudo apt update && sudo apt install ffmpeg

  4. 添加code, 运行whisper

    !whisper "audio文件" --model medium

  5. whisper帮助菜单
    !whisper -h

  6. 使用whisper medium模型，在不指定语言的情况下，转文本只花了9s

🔴 Speechnotes网页版(仅语音转文本免费)

  1. 网址

    speechnotes

  2. 实时语音转录文本是免费的，录音转文本收费
