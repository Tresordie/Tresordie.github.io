PCB Stack

1. 嘉立创提供的层压结构

2. PCB 层叠结构的影响

  1. 介质材料和铜箔特性，厚度以及叠层结构对PCB电路的性能，EMC性能都有很大的影响

3. PCB 层叠结构的主要作用

  1. 控制传输线阻抗
  2. 降低信号串扰，失真和损耗
  3. 降低电磁辐射等等

4. PCB 层叠经验及规则

  1. 每条信号线都要有最小的阻抗的回流路径，即每个信号布线层都必须有一个邻近的，紧密耦合的参考层 (最好是 GND 平面，其次是电源平面)
  2. 相邻的电源，GND 平面层保持最小间距，以提供较大的层级寄生电容 (两个层级相当于电容极板，面积大)
  3. 大面积完整的参考层 (GND 平面，电源平面) 有电磁屏蔽效果；PCB 设计中经常设计多个完整的 GND 平面或者电源平面来保护内层信号走线

5. PCB 层数评估考虑的因素

  1. 高速信号在信号完整性方面的要求，电磁兼容要求，元器件散热要求，电路板尺寸，元器件密度
  2. 传输线阻抗设计做准备，高速 PCB 布线必须控制传输线阻抗，在叠层设计和布线阶段，必须进行走线阻抗计算
  3. 影响传输线阻抗的四个参数：
  • 介电常数
  • 介质厚度
  • 铜箔厚度
  • 走线宽度、间距

6. 六层 PCB 层叠

  1. 层叠采用对称结构：保证 PCB 在受热过程中应力均匀而保持平整不发生翘曲
  2. 以下的6层板层叠，1-2注重成本(因为信号层多)，3-4注重信号完整性(信号层靠近参考层)
