2. 安装Butterfly主题时出错

  1. 安装butterfly主题时出现，extends includes/layout.pug block content include ./includes/mixins/post-ui.pug #recent-posts.recent-posts +postUI include includes/pagination.pug；为一些插件未安装

    解决方法：

    • 安装插件

      npm install --save hexo-renderer-jade hexo-generator-feed hexo-generator-sitemap hexo-browsersync hexo-generator-archive

    • 重新启动

      hexo clean && hexo g && hexo s

4. hexo generate出错

5. hexo博客迁移

  1. 安装nodejs，重启电脑，使用命令 npm -v 确认nodejs安装成功

  2. 通过nodejs安装hexo

    npm install -g hexo-cli

  3. Pandoc安装

    https://pandoc.org/installing.html

  4. hexo-render-pandoc安装

  5. 将博客文件夹拷贝到一个路径下，然后cd进入此路径下，进行npm安装

  6. 重启电脑生效

  7. 输入命令，确保编译不出错

  8. 输入命令，产生deploy下的.git文件夹

  9. 在 .deploy_git文件夹下进入.git中config文件中配置用户信息

    [user]

    ​ name = xxx
