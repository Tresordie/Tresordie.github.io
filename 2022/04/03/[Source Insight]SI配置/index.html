2. Astyle 对代码进行格式化

  1. 下载并安装 AStyle

  2. 在 Source Insight 中配置 AStyle 插件，Tools -> Custom Commands

  3. 添加新的 Command，Add -> 定义一个 Command 为 AStyle

  4. 在 Run 空白处填入以下配置：

    C:.exe --style=linux -k3 -W1 -xG -S -s4 -xb -U -p -xf -xh -xC120 -xL -H -Y -xW -w -n %f

  5. 其它勾选配置如下图配置

  6. 将 AStyle 插件功能添加到工具菜单中.

    Tool -> Custom Commands -> Menu -> Command 列表中选择定义好了的 AStyle -> Menu 列表选择 Tools -> 点击 OK -> Up/Down 来调整 AStyle 的位置
