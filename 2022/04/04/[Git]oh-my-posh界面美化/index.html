[Git]安装 oh-my-posh 美化界面(Windows)

1. PowerShell 最新版本安装

  1. 安装 PowerShell 最新版本可以很好支持一些特性

    https://github.com/PowerShell/PowerShell/releases/

2. 安装字体

  1. 安装字体支持 oh-my-posh 主题的支持

  2. 建议安装MesloLGS NF字体

    https://github.com/romkatv/dotfiles-public/blob/master/.local/share/fonts/NerdFonts/MesloLGS%20NF%20Regular.ttf

3. 安装 oh-my-posh 及 posh-git

  1. 在 Powershell 中逐条输入以下命令

    Set-ExecutionPolicy Bypass (绕过 power shell 执行策略，使其可以执行脚本文件)

    Install-Module oh-my-posh -Scope CurrentUser (oh-my-posh 安装)

    Install-Module posh-git -Scope CurrentUser (posh-git 安装)

4. 启动配置

  1. 在 PowerShell 中输入以下命令，实现启动编辑 power shell 配置文件的引擎

    if (!(Test-Path -Path $PROFILE )) { New-Item -Type File -Path $PROFILE -Force }

  2. 输入以下命令，创建配置脚本，并在脚本中添加三条基本命令

    notepad $PROFILE (创建脚本并打开)

  3. 脚本内容如下：

    Import-Module posh-git
    Import-Module oh-my-posh
    Set-PoshPrompt -Theme JanDeDobbeleer

  4. 重启 PowerShell 就可以看到 oh-my-posh 了

5. 设置主题

  1. 打开 PowerShell 之后，输入以下命令查看所有 oh-my-posh 主题

    Get-PoshTheme

  2. 可以将以下内容设置为 oh-my-posh 的脚本

6. 终端

  1. PowerShell 通过 oh-my-posh 进行美化后，可以选择其它一款 Terminal 软件实现界面更加优美化
  2. 推荐使用：FluentTerminal，Tabby
