[PyQt]PyQt5 环境搭建(Win)

1. Python3 安装

  1. 建议安装 Python3.10 之前的版本
  2. 安装 Python3.7.10

2. PyQt5 安装

3. PyQt5 tool 安装

  1. 安装 PyQt5 tool
  2. 将 PyQt5 tool 安装路径加入到系统环境变量

4. 安装 Python 模块

  1. 安装 matplotlib

    pip install matplotlib

5. Eric6 安装

  1. 安装 Qsci 模块

    pip install QScintilla

  2. 下载 Eric6，进入到解压缩的文件夹，进行源码安装

    python ./install.py

  3. Eric6 的汉化：

    • 下载汉化文件夹到 i18n 文件夹下
    • 将 install-i18n.py 放到 eric6 的根目录下并运行

6. 插件安装

  1. 安装 jedi 插件(自动补齐)

    pip install jedi

  2. Eric6 安装(手动安装)

    • 打开 Eric6，菜单栏选择 extra -- 插件 -- 安装插件 -- 选择下载的 jedi 压缩文件进行安装
