BLDC 六步驱动(基于 STM32)

2. 六步驱动的基本思路

  1. 三相霍尔传感器连接至通用定时器，通用定时器捕获霍尔传感器的状态变化，作用是确定磁铁转子的起始位置及当前转子处于六步中的哪一步
  2. 高级定时器用于驱动三相 BLDC 电机，配置为输出比较方式，互补输出 PWM 的方式驱动，驱动方式有五种：PWM_ON, ON_PWM, H_PWM-L_ON, H_ON-L_PWM, H_PWM-L_PWM
  3. 高级定时器换相输出时进行预装载，通过 COM 事件进行预装载更新实现换相
  4. 通用定时器每次捕获到传感器状态变化后，就会这样持续驱动电机旋转
  5. 还可以同步外部触发方式来对电机速度进行调整

4.3 电机堵转检测

  1. 输入捕获的通用定时器开启了定时器更新中断及双边沿触发中断，定时器更新计数时长一定要大于边沿触发的间隔时长
  2. 在电机不堵转的情况下，通用定时器可以持续不断的捕获到边沿触发(由于捕获成功后，会复位计数器，故而永远不会进入技术更新中断)，
  3. 当电机堵转时，由于无法捕获到霍尔传感器的电平变化，会进入通用定时器的更新中断，在更新中断中进行计数，当计数达到 2 次后，认为堵转事件发生，此时可以关闭电机驱动

5. 通过 hall sensor 值确定换相控制

5.1 通过 BLDC 电机的规格书来确定

  1. BLDC 规格书中列出了 hall sensor 真值表与三相电压控制时序之间的对应关系
  2. 注意真值表是 BLDC 逆时针转动时的对于关系表
  3. 逆时针的 hall_c, hall_b, hall_a 为 0 0 1 时，如果是顺时针旋转时，对应上图中的 hall_c, hall_b, hall_a 为 1 1 0,  那么此时顺时针旋转需要加电 U+, W-

5.2 通过分析来确定

  1. 顺时针旋转时，当霍尔传感器 hall_c, hall_b, hall_a 值为 0 0 1 时，其上一个状态的霍尔传感器值为 hall_c, hall_b, hall_a 值为 0 1 1，下一个状态的霍尔传感器值为 hall_c, hall_b, hall_a 值为 1 0 1
  2. 下图中转子 S 极起始位置转到红色区域某个位置时，才触发了霍尔传感器值为 0 0 1，此时需要对电机驱动进行换相，让电机继续顺时针运转直到蓝色笔迹处的 N, S 极；相当于下面图片中的磁极转换的第 5 步向第 6 步转换，此时需要在第 5 步的基础上，改变换相控制输出设定为第 6 步中的方式，为  U+, W- ；这与上面电机规格书中真值表对应关系一致

5.3 固件代码中换相

  1. 固件代码中根据霍尔传感器值进行换相的配置就是按照其来的
  2. 固件代码中也指定了 BLDC 电机正向及反向转时的不同配置设定
