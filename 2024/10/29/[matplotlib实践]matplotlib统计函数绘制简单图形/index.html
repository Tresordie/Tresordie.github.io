matplotlib统计函数绘制简单图形

发表于2024-10-29|更新于2024-10-29|Python
|阅读量:

1. 函数 bar() -- 绘制柱状图

  • plt. bar(x, y): Matplotlib 中用于绘制垂直条形图的函数。它可以用于显示不同类别的数据大小或频率，条形图的高度表示值的大小

    x：条形图的类别（通常是名称列表或数值）。

    height：每个类别对应的值，即条形的高度。

    color：条形颜色，可以是单个颜色，也可以是颜色列表。

    width：条形的宽度，控制条形的粗细，默认值为 0.8。

    align：控制条形的位置，默认是 'center'，可以设置为 'edge' 以对齐边缘。

    xerr 和 yerr：分别用于设置 x 和 y 方向的误差条。

2. 函数 barh() -- 绘制条形图

  • plt.barh(x, y): Matplotlib 中用于绘制水平条形图的函数。它类似于 plt.bar()，但是条形图是水平显示的，适用于需要比较不同类别大小的场景

    y：条形图的类别（通常是名称列表）。

    width：每个类别对应的值，即条形的宽度。

    color：条形颜色，可以是单个颜色，也可以是颜色列表。

    height：条形的高度，用于控制条形的粗细。

    align：控制条形的位置，默认是 'center'，可以设置为 'edge' 以对齐边缘。

    xerr 和 yerr：分别用于设置 x 和 y 方向的误差条。

3. 函数 hist() -- 绘制直方图

  • plt.hist(x): 在 x 轴上绘制定量数据的分布特征

    x：输入数据（必需）。

    bins：设置分组（bin）的数量，可以是整数、序列或字符串，如 bins=30 表示30个区间。

    range：一个包含两个元素的元组，用于设置数据的最小值和最大值范围（例如 (min, max)）。

    density：布尔值，若为 True，直方图将显示密度而非频率，使得每个条形的面积为 1。

    color：条形颜色。

    edgecolor：条形边缘颜色。

    alpha：设置透明度（0-1）

4. 函数 pie() -- 绘制饼图

  • plt.pie(x): 是 Matplotlib 中用于绘制饼图的函数，它根据一组数据在整个集合中的比例来生成一个圆形图表。饼图常用于表示数据在不同类别中的比例分布

    x：数据序列，表示每个部分的大小。

    labels：用于指定每部分的标签。

    colors：每个部分的颜色，可以是颜色名称、Hex 值等。

    autopct：格式化每个部分的百分比显示方式，例如 '%1.1f%%' 表示保留一位小数的百分比。

    startangle：设置饼图的起始角度，以度为单位。例如，startangle=140 表示从 140° 开始。

    explode：设置将某些部分“突出”显示的效果，值越大越“突出”，如 [0, 0.1, 0, 0]。

5. 函数 polar() -- 绘制极线图

  • plt.polar(theta， r): 在极坐标轴上绘制折线图

    theta：角度数据，通常以弧度表示。

    r：半径数据。

    color：线条或填充区域的颜色。

    linewidth：线条宽度。

    linestyle：线条样式，如 '-'、'--'、':' 等。

6. 函数 scatter() -- 绘制气泡图

  • plt.scatter(x): Matplotlib 中用于绘制散点图的函数。散点图用于显示两个变量之间的关系或分布，可以直观地观察到数据点的分布情况

    x：X 轴数据（必需）。

    y：Y 轴数据（必需）。

    s：每个点的大小，可以是单个值或与 x 和 y 相同长度的数组。

    c：每个点的颜色，可以是单个颜色、颜色列表或与 x 和 y 相同长度的数组。

    alpha：点的透明度，值在 0 到 1 之间，0 表示完全透明，1 表示不透明。

    marker：点的形状，常用的有 'o'（圆形）、's'（方形）、'^'（三角形）等。

7. 函数 stem() -- 绘制棉棒图

  • plt.stem(x, y): Matplotlib 中用于绘制“茎叶图”的函数，常用于表示离散数据点的情况。茎叶图由垂直线段（茎）和水平线段（叶）组成，直观地显示数据的分布

    x：指定棉棒 x 轴基线上的位置 。
    y：绘制棉棒的长度 。

    linefmt：指定茎的线条格式（如颜色、样式等）。

    markerfmt：指定叶的标记格式（如颜色、形状等）。

    basefmt：指定基线的样式，可以设置为不同的颜色或样式。

    use_line_collection：布尔值，如果为 True，将使用 LineCollection 来加速绘图。

8. 函数 boxplot() -- 绘制箱线图

  • plt.boxplot(x): Matplotlib 中用于绘制箱形图（Box Plot）的函数。箱形图用于可视化数据的分布情况，特别是用于展示数据的中心位置、散布程度以及异常值。它能够直观地显示数据的五个数字摘要（最小值、第一四分位数、中位数、第三四分位数和最大值）

    x：数据数组或数组的列表，通常为一维数组或多个一维数组的列表。

    vert：布尔值，决定箱形图是垂直（True）还是水平（False），默认值为 True。

    patch_artist：布尔值，若为 True，则使用填充的样式（每个箱子可以用不同的颜色）。

    notch：布尔值，若为 True，则绘制带缺口的箱形图，便于比较中位数。

    whis：用于设置“胡须”的长度，可以是数字（代表倍数）或百分比，默认值为 1.5，表示超出1.5倍四分位距的点将被视为异常值。

    labels：设置每个箱形图的标签。

9. 函数 errorbar() -- 绘制误差棒图

  • plt.errorbar(x,y,yerr,xerr): Matplotlib 中用于绘制带误差条的图形的函数。它可以显示数据点的同时，加入误差信息，以帮助用户了解数据的不确定性或变异性

    x：X 轴数据（必需）。

    y：Y 轴数据（必需）。

    yerr：Y 轴方向的误差，可以是标量（所有点相同的误差）或与 Y 数据相同长度的数组。

    xerr：X 轴方向的误差，类似于 yerr。

    fmt：数据点的格式字符串，例如 'o' 表示圆点，'s' 表示方形等，通常用于设置数据点的样式。

    color：设置数据点和误差条的颜色。

    capsize：设置误差条端点的帽子大小，单位为点。

    label：图例标签，用于标识数据系列。
